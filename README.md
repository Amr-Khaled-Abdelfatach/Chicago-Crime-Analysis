# Chicago-Crime-Analysis
End-to-end big-data analytics project on Chicago crime data: PySpark ETL/EDA, arrest-prediction classification (AUC 0.88) and K-Means clustering, plus a real-time Kafka → Flink streaming pipeline for windowed aggregations.
**The DataSet Link**
https://data.cityofchicago.org/Public-Safety/Crimes-2001-to-Present/ijzp-q8t2/about_data
